# Student Performance Analysis Using SQL

## Project Overview

This project analyzes the relationship between student behaviors and final academic performance using SQL.

### Tools

- MySQL
- DBeaver
- SQL
- Excel

## Dataset

The dataset contains student information including:

- Gender
- Study time
- Absences
- Failures
- Internet access
- Higher education intention
- Final grade (G3)

## Key Findings

### 1. Study Time

学习时间增加时，平均成绩整体提高。

| Study Time | Average Score |
|------------|---------------|
| 1 | 9.95 |
| 2 | 10.15 |
| 3 | 11.40 |
| 4 | 11.26 |

### 2. Failures

挂科次数越多，最终成绩越低。

| Failures | Average Score |
|-----------|--------------|
| 0 | 11.24 |
| 1 | 7.96 |
| 2 | 6.24 |
| 3 | 5.69 |

### 3. Higher Education

有升学意愿学生平均成绩：10.57

无升学意愿学生平均成绩：6.80

### 4. Internet Access

有网络学生平均成绩更高。

### 5. Romantic Relationship

无恋爱关系学生平均成绩高约1.2分。

## Conclusion

Among all analyzed factors, previous failures and higher education intention showed the strongest relationship with academic performance.

Students with clear academic goals generally achieved better final grades.
